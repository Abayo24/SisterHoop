SisterHoop
Introduction
SisterHoop is a web application designed to connect female basketball players, offering a supportive environment where they can find local courts, join pickup games, and connect with other players. The platform aims to enhance the basketball experience for women by providing a space free from competition and overshadowing. üöÄüèÄ
